# WEI IN SIGHT — Rebuild Spec Pack

This folder contains the build blueprint for the **WEI IN SIGHT** website rebuild.

## What this pack is for
This pack is written so Antigravity and any connected design or coding agents can work from one clear source of truth without drifting off concept, rebuilding from scratch unnecessarily, or flattening the poetic identity of the site.

## Build principle
The site is **not** a generic portfolio.  
It is a **cinematic personal universe** where the homepage uses a **human-body navigation metaphor** and each creative discipline belongs to a meaningful body anchor.

## Critical product decision
Section destination behavior uses **Option B**:

- The landing page is the immersive gateway.
- Clicking a submenu item routes to a **dedicated destination page**.
- The landing page is not the final container for all content.
- The immersive homepage should remain light, memorable, and elegant.
- SEO, deep content, shareability, and long-term scalability are handled by dedicated section pages.

## Stage workflow
Use the stage prompts in order.

Important:
- Each stage must **build on the existing project state**.
- No later stage may restart the project from scratch.
- Preserve completed architecture, routing, and design language unless a change is explicitly required.
- Keep naming, spacing rhythm, color logic, motion language, and body-anchor metaphor consistent across all stages.

## Tone of the final website
Elegant. Poetic. Premium. Cinematic. Human. Futuristic.  
Not gimmicky. Not noisy. Not game-like. Not clinical.
