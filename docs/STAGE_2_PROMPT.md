# Stage 2 Prompt — Landing Page Layout, Navigation UX, and State Logic

Use this prompt in Antigravity for Stage 2.

```md
Continue from the existing WEI IN SIGHT project state.
This is Stage 2 only.

Important rules:
- Do not restart the project.
- Do not replace routing created in Stage 1.
- Do not rebuild the site from scratch.
- Do not invent new primary sections.
- Do not implement final 3D mesh behavior yet.
- Build on the Stage 1 foundation exactly as it exists.

Goal:
Build the landing page experience structure and interactive UI shell without full final 3D implementation yet.

Desktop homepage layout:
- left panel for branding and navigation
- right panel for immersive body-scene area

Left panel content:
- wordmark: WEI IN SIGHT
- subtitle: The creative atlas of Jacky Ho
- short intro line
- vertical main nav:
  - Sight — Visual Arts
  - Sound — Music & Audio
  - Touch — Mixed Media & Process
  - Voice — Poems & Lyrics
  - Dream — Novel & Story World
  - Heart — About / Philosophy / Journey
  - Pulse — Buy Art / Commissions / Contact
- subtle footer links for Instagram / Email / Collect or Inquire

Right panel content:
- placeholder or simplified scene area representing the future body scene
- visible anchor states that correspond to each section
- structure ready for camera-focus logic and submenu emergence

Interaction goals:
1. Hovering a main nav item should preview its associated anchor state.
2. Clicking a main nav item should shift the homepage into a focused state for that body anchor.
3. Submenu items should appear as if they emerge from the selected anchor area.
4. Submenu items should route to dedicated destination pages.
5. Keep all interactions elegant and restrained.

Body-anchor mapping:
- Eyes = Sight
- Ears = Sound
- Hand/Fingertips = Touch
- Throat = Voice
- Forehead/Temple = Dream
- Chest/Heart = Heart
- Wrist = Pulse

Submenus:
- Sight: Paintings, Photography, Sculpture, Collections
- Sound: Streaming Platforms, Original Songs, Audio-Visual Work, Music Archive
- Touch: Mixed Media, Material Experiments, Fabric/Texture Work, Studio Process
- Voice: Poems, Lyrics, Spoken Word, Journal Fragments
- Dream: The Novel, Story World, Excerpts, Behind the Story
- Heart: About Me, Philosophy, Journey, Exhibitions/Features
- Pulse: Buy Art, Available Works, Commissions, Contact

Stage 2 deliverables:
- polished split-screen homepage layout
- left navigation with hover and active states
- intro copy placement
- submenu reveal behavior
- focused-state logic for selected section
- responsive homepage adaptation
- simplified mobile version preserving the metaphor
- transition patterns that will carry into Stage 3

Design direction:
- dark, cinematic, premium
- neon pink accent
- white typography
- restrained magenta/violet depth if useful
- elegant spacing
- no clutter

Do not:
- turn the homepage into a long scrolling generic landing page
- bury the navigation metaphor
- use loud cyberpunk styling
- add gimmicky animation

Use Google Stitch MCP for:
- layout balance
- navigation spacing
- hierarchy refinement
- responsive behavior
- component polish

Acceptance criteria:
- homepage feels intentional and premium before the final 3D build
- all main and submenu states are understandable
- mobile version is coherent
- Stage 3 can replace the placeholder scene with the real body scene without redesigning the rest of the homepage

At the end, summarize:
- what homepage states were built
- how submenu logic works
- how the layout responds on mobile
- what is ready for Stage 3
```
