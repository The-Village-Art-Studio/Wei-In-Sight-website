# Stage 4 Prompt — Dedicated Section Pages and Editorial Templates

Use this prompt in Antigravity for Stage 4.

```md
Continue from the existing WEI IN SIGHT project state.
This is Stage 4 only.

Important rules:
- Do not restart the site.
- Do not rebuild the homepage from scratch.
- Do not replace the body-navigation concept.
- Build on the route structure and templates already established.
- Preserve design language, spacing rhythm, color logic, and tone.

Goal:
Build the dedicated destination pages for each primary section and subsection route.

This project uses Option B:
homepage = gateway
destination pages = deep content

Primary sections:
- Sight
- Sound
- Touch
- Voice
- Dream
- Heart
- Pulse

Build or refine:
- main section landing pages
- subsection templates
- reusable editorial components
- CMS/content rendering for each content type

Section-specific page intentions:

Sight:
- editorial, gallery-like
- strong image hierarchy
- series/collection presentation

Sound:
- music and audio focused
- embeds and platform links
- atmosphere without clutter

Touch:
- tactile, material, process-driven
- close-up imagery and studio feel

Voice:
- intimate, text-led, quiet
- poems, lyrics, spoken language, journal fragments

Dream:
- cinematic and narrative
- novel, story world, excerpts, behind-the-story material

Heart:
- personal and reflective
- biography, philosophy, journey, exhibitions/features

Pulse:
- collector-facing and practical
- available works, commissions, contact, inquiry

Required deliverables:
- refined section page templates
- local subnavigation for each section
- reusable content blocks
- editorial hierarchy that matches each section’s mood
- SEO-ready route pages
- CTA patterns where appropriate

Do not:
- make each section look like a totally different website
- abandon the core brand language
- overload sections with decorative effects
- let templates become inconsistent

Use Google Stitch MCP for:
- editorial layout refinement
- responsive behavior
- content hierarchy
- spacing polish
- page-level consistency audit

Acceptance criteria:
- each section has a distinct mood but shared design DNA
- subsection pages are scalable
- content can be updated through the admin/content layer
- the transition from homepage to destination pages feels coherent

At the end, summarize:
- templates created/refined
- reusable blocks created
- content assumptions
- what remains for Stage 5
```
